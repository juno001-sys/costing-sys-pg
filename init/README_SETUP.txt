==========================================
 Kurajika CMS - Costing System
 初期セットアップ & 起動手順
==========================================

■ 初回セットアップ
------------------------------------------
cd ~/kurajika_cms/costing_sys/init
chmod +x init_costing.sh run_costing.sh
./init_costing.sh

■ 起動方法（2回目以降）
------------------------------------------
cd ~/kurajika_cms/costing_sys/init
./run_costing.sh

ブラウザでアクセス：
http://127.0.0.1:5000/

止める時：
Ctrl + C
