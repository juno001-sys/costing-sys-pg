## 📘 ヘルプ2章の追加 / Two New Help Chapters

### 2-1. 休業日の設定と仕入先との関係 (`/help/concept_holidays`)
オペレーターが迷いやすい「2種類の休業日の違い」を整理。

- **店舗休業日** vs **仕入先固有の休業日** の比較カード
- 重要スイッチ **『店舗休業に連動する』フラグ** の判断ガイド
- **4パターンの早見表**（店舗休業＋連動ON/OFF、仕入先盆休み、連休ギャップ警告）
- 3ステップの設定手順（カレンダー → フラグ確認 → 仕入先独自休業）
- よくある質問 3つ

### 2-2. 発注サポートの使い方 (`/help/concept_order_support`)
「何を見ればよいか」より「なぜこの表示になるか」を説明。

- **データフロー図**：5つの入力（仕入先／品目／棚卸し／仕入れ／店舗カレンダー）→ 発注サポート
- **2つの表示モード**（カード / 表形式）をスクリーンショット付きで比較
- **在庫状態の判定式**：`現在庫 < 発注目安 → 不足` / `< ×1.5 → 残少` / `≥ ×1.5 → 充足`
- **おすすめワークフロー**：棚卸し → 店舗選択 → 〆切順 → 発注 → 納品記録（5ステップ）
- **将来ロードマップ**（NOW / NEXT / LATER の色分けタグ付き）
  - Phase 0：静的判定（現在）
  - Phase 1：日次需要の手入力 → 未来予測
  - Phase 1.5：補正係数（revise_ratio）
  - Phase 2：予約システム連携
  - Phase ?：発注書の自動生成（メール／FAX）

両章とも `🧭 基本の考え方` セクションに追加されました。
